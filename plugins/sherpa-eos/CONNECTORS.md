# Connectors

The **sherpa-eos** plugin runs against an EOS platform via MCP.

| Category | Used for |
|----------|----------|
| EOS platform (**Strety**) | Reading and writing Rocks, Scorecard metrics, Accountability Chart, V/TO, and L10 meeting data. |

## Setup

Connect the **Strety** MCP connector before using the skills. Without it, the
skills have no system to read from or write to. The plugin does not bundle any
credentials — authentication is handled by your own connector.
